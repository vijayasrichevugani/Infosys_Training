Step 1: Enable BQ API
 
Step 2: Set up Public Dataset 
https://console.cloud.google.com/bigquery?p=bigquery-public-data&page=project

Step 3:  Query Public Dataset 
SELECT
  name, gender,
  SUM(number) AS total
FROM
  `bigquery-public-data.usa_names.usa_1910_2013`
GROUP BY
  name, gender
ORDER BY
  total DESC
LIMIT
  10
